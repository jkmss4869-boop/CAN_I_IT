# Guess The Noun - ESL Duo Game (Grade 7)

Game suy luận logic 2 người chơi dành cho lớp ESL Online.

## Cấu trúc thư mục:
- `index.html`: Giao diện chính chứa Main Menu, Màn hình chơi (P1 & P2), và Màn hình chiến thắng.
- `style.css`: File tạo phong cách hoạt họa hiện đại, dễ thương phù hợp học sinh lớp 7.
- `script.js`: Xử lý logic đảo cột (Mirrored layout) và kiểm tra kết quả Yes/No toán học.

## Hướng dẫn đưa lên GitHub Pages:
1. Tạo một repository mới trên GitHub (Ví dụ: `guess-the-noun-game`).
2. Upload cả 3 file (`index.html`, `style.css`, `script.js`) lên Repo đó.
3. Vào tab **Settings** -> chọn mục **Pages** ở menu bên trái.
4. Tại phần **Branch**, chọn `main` rồi bấm **Save**.
5. Đợi khoảng 2 phút, GitHub sẽ cấp cho bạn một đường link dạng `https://<username>.github.io/guess-the-noun-game/`.
6. Gửi link cho học sinh, bạn chọn màn hình Player 1, học sinh chọn Player 2 và cùng mở micro lên chơi!
